ChatGPT is OpenAI's AI assistant, powered by the GPT-4o model family. As a day-to-day tool it is one of the most widely used AI assistants in the world. For technical teams the OpenAI API unlocks multimodal capabilities — GPT-4o can understand and describe images, making it useful for exhibits that combine visual input with natural language output.

## OpenAI API

The OpenAI API provides programmatic access to GPT-4o and other models. Key capabilities relevant for museum work:
- **Text generation:** Conversational AI, guided tours, interpretive text
- **Vision:** Describe or analyse an image input (useful for camera-based interactive exhibits)
- **Whisper:** OpenAI's speech-to-text model; transcribe spoken visitor questions in real time
- **TTS:** Text-to-speech; generate voiced responses for accessibility or immersive audio guides

## ChatGPT vs Claude

Both are powerful general-purpose LLMs. Key practical differences:
- **ChatGPT** has broader public recognition; visitors are more familiar with it
- **Claude** has a larger context window and is generally preferred for coding and long documents
- **OpenAI API** has a richer ecosystem of third-party integrations and community examples

## Use Cases

- Conversational AI guides that answer visitor questions about artworks or exhibitions
- Voice-activated information kiosks using Whisper for speech input + TTS for audio output
- Automated captioning or description of artworks via Vision API
- Generating accessible text alternatives for complex exhibition content
