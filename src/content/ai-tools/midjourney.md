Midjourney is an AI image generation service known for producing the most aesthetically refined and art-directed results of any current image model. Unlike Stable Diffusion (open-source, local) or DALL-E (via ChatGPT), Midjourney is a closed hosted service with a strong visual identity — it consistently produces work that reads as intentional and art-directed rather than generically AI-generated.

## Why Midjourney Stands Out

The output quality and coherence of Midjourney images is generally higher than alternatives for artistic and conceptual work — particularly for architectural visualisation, concept art, and atmospheric scenes. It is used extensively by design studios and creative agencies as a rapid concept generation and moodboarding tool.

## Access

Midjourney operates via Discord — users submit prompts in a Discord server or via direct message to the Midjourney bot. A web interface is available in beta. There is no open API (unlike Stable Diffusion or DALL-E), which limits integration into pipelines.

## Use Cases for Museum Work

- Rapid moodboarding and concept visualisation for proposed installations
- Generating reference imagery for set design, lighting direction, and atmosphere
- Creating unique imagery for promotional materials and exhibition graphics
- Exploring visual directions without commissioning illustration or photography

## Prompting Tips

- Use `--ar` flag for aspect ratios: `--ar 16:9`, `--ar 1:1`
- Reference art styles, photographers, and film directors for directed aesthetics
- `--stylize` parameter controls how strongly Midjourney applies its own aesthetic (lower = more literal)
