Claude is Anthropic's AI assistant — a large language model capable of writing, coding, analysis, and conversation. For the museum tech team it is useful in two distinct ways: as a day-to-day assistant for writing, documentation, and problem-solving; and as an API for building interactive exhibits that respond to natural language.

## Claude Code

Claude Code is Anthropic's AI coding assistant that works directly in the terminal alongside your codebase. It can read, write, and edit files; run commands; understand large codebases; and work through multi-step technical problems autonomously. It is the tool used to build and maintain this taxonomy.

Useful for the tech team for:
- Rapidly prototyping and iterating on installation software
- Understanding unfamiliar code or debugging complex systems
- Generating boilerplate, configurations, and documentation
- Working through technical problems in natural language

## Claude API for Interactive Exhibits

The Claude API allows you to build exhibits where visitors converse naturally with an AI character, guide, or assistant. The API is fast enough for real-time interaction and supports system prompts for giving the AI a specific persona, knowledge base, or set of constraints.

## Use Cases

- Day-to-day: writing exhibition text, documentation, research, problem-solving
- Development: Claude Code for coding assistance on installation software
- Interactive exhibits: AI-powered interpretive guides, question-answering kiosks
- Content generation: drafting accessible descriptions of artworks and technologies
