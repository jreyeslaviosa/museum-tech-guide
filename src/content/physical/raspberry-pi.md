Raspberry Pi is a low-cost, credit-card-sized single-board computer running Linux. Unlike Arduino (a microcontroller), Raspberry Pi is a full computer capable of running a desktop OS, web browser, and media playback software. Widely used in museum installations as a lightweight, fanless media player or kiosk controller.

## Use Cases

- Driving a dedicated screen or kiosk display in a gallery (Raspberry Pi as media player)
- Running small interactive installations without a full desktop PC
- Hosting Node.js or Python scripts for sensor-driven interactive experiences
- Low-cost show trigger controller communicating over OSC, MQTT, or serial

## Raspberry Pi vs Arduino

| | Raspberry Pi | Arduino |
|---|---|---|
| Type | Single-board computer | Microcontroller |
| OS | Linux | None (bare metal) |
| Language | Python, JS, etc. | C/C++ |
| Best for | Media playback, networking | Hardware I/O, real-time control |
