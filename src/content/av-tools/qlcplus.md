QLC+ (Q Light Controller Plus) is a free, open-source lighting control software for controlling DMX fixtures. It runs on macOS, Windows, and Linux and supports USB DMX interfaces (Enttec), ArtNet output, and a wide fixture library. A good starting point for learning DMX programming.

## Use Cases

- Controlling stage lighting rigs for small to medium productions
- Rapid prototyping of lighting looks and DMX channel mappings before using a full console
- Driving ArtNet universes to pixel controllers from a standard laptop
- Educational tool for learning DMX patching and fixture addressing
