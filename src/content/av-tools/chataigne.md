Chataigne (French for chestnut) is a free, open-source show control and protocol bridge application. It connects almost any protocol to any other — OSC, MIDI, DMX, ArtNet, HTTP, Serial, WebSocket, and more — using a visual state machine editor. Where TouchDesigner is the tool for real-time generative content, Chataigne is the tool for routing signals and orchestrating a show's system integration.

## What Makes It Useful

Interactive installations often involve many separate systems that need to talk to each other: a sensor system, a media server, a lighting controller, a sound system. Chataigne acts as the hub — mapping messages between them, transforming values, and triggering sequences. No coding required.

## Use Cases

- Routing OSC messages from a depth camera system to trigger QLab audio cues and DMX lighting changes simultaneously
- Building a state machine that changes the installation's behaviour based on visitor presence (idle → active → cooldown)
- Bridging MIDI controller input to ArtNet lighting output without writing code
- Rapid prototyping of show control logic before committing to a custom-coded solution
