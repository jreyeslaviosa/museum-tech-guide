Madrix is a professional LED lighting control software for large-scale pixel mapping, dynamic effects, and synchronised LED installations. It specialises in generating and mapping content directly to LED matrices, strips, and panels via ArtNet or sACN — without needing a separate media server.

## Use Cases

- Generating procedural effects (colour washes, chases, particle effects) across large LED arrays
- Pixel mapping video content onto non-linear LED arrangements
- Creating synchronised multi-zone LED light shows from a single software controller
- Driving architectural LED installations in permanent gallery or building facade applications
