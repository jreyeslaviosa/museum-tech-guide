Watchout is a professional multi-display production system from Dataton for driving large-scale projection mapping, video walls, and complex multi-screen shows. It uses a dedicated show computer model where a single control machine manages multiple output computers over a local network.

## Use Cases

- Driving multi-projector blended panoramic displays in permanent gallery installations
- Synchronising video across multiple LED panels and projection surfaces
- Warping and edge-blending projector arrays onto irregular architectural surfaces
- Running cue-based timeline shows triggered by show control systems
