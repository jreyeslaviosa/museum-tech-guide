Birddog is an Australian company that builds hardware and software centred on the NDI video-over-IP ecosystem. Their products convert HDMI, SDI, and other video signals to and from NDI, making it practical to route any video source across a network without capture cards or long cable runs.

## Key Products

- **Birddog P200 / P400:** PTZ cameras with built-in NDI output; useful for live capture in interactive installations
- **Birddog PLAY:** Software media player that outputs to NDI — play a video file from one machine and receive it as a live NDI stream anywhere on the network
- **Birddog QUAD:** Four-channel HDMI-to-NDI encoder; bring four HDMI sources onto the network simultaneously
- **Birddog STUDIO:** Cloud-based production platform for remote productions

## Birddog PLAY

The user mentioned Birddog PLAY specifically. It is a free Windows/macOS media player that plays back video files and broadcasts them as an NDI stream. Useful for getting pre-rendered content onto the network cleanly without routing physical cables.

## Use Cases

- Converting a laptop's HDMI output to NDI for distribution to multiple screens over a network
- Using Birddog PLAY to send pre-rendered show content as an NDI feed to a media server
- PTZ camera input into TouchDesigner or a media server via NDI for live capture installations
- Eliminating long HDMI cable runs across a venue by routing video over the existing network
