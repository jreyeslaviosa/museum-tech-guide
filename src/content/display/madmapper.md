MadMapper is an intuitive projection mapping and LED mapping tool designed for artists, VJs, and installation creators. It handles both video projection onto 3D surfaces and pixel mapping for LED grids, strips, and matrices via DMX/ArtNet.

## Use Cases

- Quick projection mapping onto objects and architectural surfaces without complex media server setup
- Mapping generative content from TouchDesigner or Resolume onto irregular surfaces via Syphon/Spout
- Pixel mapping LED installations with the built-in LED fixture library
- Creating interactive mapped content with its built-in scripting (GEM scripts)
