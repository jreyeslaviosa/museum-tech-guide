NDI (Network Device Interface) is a royalty-free IP video protocol developed by NewTek/Vizrt that allows video sources to be shared across a local network at low latency. Any NDI-enabled device or application can send and receive video streams to/from any other device on the same network.

## Use Cases

- Sending video from one computer to another without capture cards or physical cables
- Routing generative content from TouchDesigner to a media server over the network
- Monitoring live video feeds on any machine in the studio
- Building distributed multi-computer show systems where each machine feeds a different output
