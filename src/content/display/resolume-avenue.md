Resolume Avenue is the VJ/media server version of Resolume — it handles real-time video mixing, effects, and playback without Arena's advanced projection mapping features. Avenue is the right tool when you need a full-featured media server for live performance, club visuals, or gallery installations where output mapping is simple (one or a few flat screens). If you need soft-edge blending, warping, and multi-projector mapping, you need Arena instead.

## Avenue vs Arena

| | Avenue | Arena |
|---|---|---|
| Real-time video mixing | ✓ | ✓ |
| Effects & generative | ✓ | ✓ |
| Syphon / Spout input | ✓ | ✓ |
| Output warping | Basic | Advanced |
| Soft-edge blending | No | Yes |
| DMX input | ✓ | ✓ |
| Price | ~$400 | ~$700 |

## Key Features

- **Clip-based timeline:** A grid of video/effect clips triggered in real time — similar workflow to Ableton Live's session view
- **FX chain:** Per-layer and per-clip effects (colour, distortion, generative) that process in real time
- **Syphon/Spout:** Receive frames from TouchDesigner, Unity, or any GPU-rendered source over the GPU bus
- **DMX input:** Map external lighting console channels to any parameter for show-synchronised control
- **OSC control:** Full parameter control via OSC from any show control system

## Use Cases

- Gallery installations with a single screen or simple multi-screen output
- Live performance visual systems driven by a VJ or show control
- Receiving TouchDesigner output and mixing/compositing it with pre-rendered clips
- Generative real-time video for events and installations where Arena's mapping is unnecessary overhead
