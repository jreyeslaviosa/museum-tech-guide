Unreal Engine is a professional real-time 3D engine used in games, film, and increasingly in live event and museum production. Its nDisplay system makes it the industry standard for LED volume / virtual production stages, and it is increasingly used for permanent immersive installations that require photorealistic real-time rendering.

## Why Unreal for Installations

Unlike TouchDesigner or Notch which are effects-oriented, Unreal is a full 3D game engine with a complete scene graph, physically-based rendering (PBR), ray tracing, and a visual scripting system (Blueprints). For museum spaces that want photorealistic environments, interactive narrative experiences, or architectural-scale 3D, Unreal is the most capable real-time option.

## nDisplay

nDisplay is Unreal's multi-display cluster system — it allows one Unreal scene to render across multiple outputs (projectors, LED panels, curved screens) with hardware-synchronised frames. This is the same technology used in major LED volume studios (The Volume on The Mandalorian) and scales down to museum installations.

## Key Features for Museum Use

- **Blueprints:** Visual scripting system — no C++ needed for interactive logic
- **Lumen / Nanite:** Fully dynamic global illumination and virtualised geometry at real-time frame rates
- **Media I/O:** Live video capture via NDI, Blackmagic, and AJA into scenes
- **OSC / DMX input:** Receive show control signals, map to scene parameters
- **Python scripting:** Automation via the editor and runtime

## Use Cases

- LED volume stages for content production and live events
- Photorealistic museum walk-through experiences driven by visitor interaction
- Architectural visualisation rendered live in an installation space
- Multi-projector blended panoramic environments with 3D content
