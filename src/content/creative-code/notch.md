Notch is a GPU-accelerated node-based tool for creating real-time graphics, effects, and interactive content. It excels at complex particle systems, fluid simulations, and procedural visuals — and outputs as a plugin into media servers like Disguise, disguise, and Green Hippo.

## Use Cases

- Creating physically-based particle and simulation effects for projection mapping
- Generating real-time reactive visuals that respond to audio or sensor data
- Embedding complex visual effects into media server workflows (Disguise d3)
- Building interactive content where rendering speed is critical
