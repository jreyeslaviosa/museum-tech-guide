Spout (Windows) and Syphon (macOS) are frameworks that allow real-time video frames to be shared between applications on the same computer via the GPU — with zero latency and zero CPU copy overhead. They are the standard way to pipe the output of TouchDesigner, Unity, or Blender into Resolume, MadMapper, or any other application that needs a live video feed.

## Why This Matters

Without Spout/Syphon, passing video between applications requires encoding to a codec, sending over a virtual video cable or NDI, and decoding — adding latency and CPU load. Spout/Syphon share a GPU texture directly: the sending application writes to shared memory on the GPU; the receiving application reads from the same memory. Latency is one frame (16ms at 60fps).

## Spout vs Syphon vs NDI

| | Spout | Syphon | NDI |
|---|---|---|---|
| Platform | Windows only | macOS only | Cross-platform |
| Transport | GPU shared memory | GPU shared memory | Network (CPU encode) |
| Latency | ~1 frame | ~1 frame | 1–3 frames |
| CPU overhead | Near zero | Near zero | Significant |
| Cross-machine | No | No | Yes |

Use Spout/Syphon for same-machine connections; use NDI when you need to cross to another computer on the network.

## Common Pipelines

```
TouchDesigner → Spout out → Resolume (Spout in source)
Unity → Syphon plugin → MadMapper (Syphon input)
Blender EEVEE → Spout → Watchout input
```

## Software Support

**Sends Spout/Syphon:** TouchDesigner, Unity (plugin), Unreal (plugin), Blender (plugin), Processing, openFrameworks, most creative coding environments

**Receives Spout/Syphon:** Resolume Arena/Avenue, MadMapper, Millumin, OBS, vMix, Disguise (via plugin)

## Use Cases

- Using TouchDesigner as a generative video source feeding into Resolume for effects and output mapping
- Sharing a Unity game engine render into MadMapper for projection mapping
- Compositing multiple generative sources in Resolume where each source is generated by a different application
- Low-latency live camera processing in TouchDesigner fed to a media server output
