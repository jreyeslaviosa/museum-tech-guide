p5.js is a JavaScript library that brings Processing's visual coding philosophy to the web browser. It runs on any device with a browser — no installation, no hardware requirements — making it the most accessible way to create interactive graphics, generative art, and data visualisations for web-embedded museum exhibits.

## Why p5.js

The browser is now a legitimate creative platform. p5.js sketches run on any screen — from a touchscreen kiosk to a visitor's phone — and integrate naturally with web technologies: HTML, CSS, video, camera, microphone, and WebGL. The p5.js editor at editor.p5js.org lets anyone start coding in a browser with zero setup.

## Use Cases

- Browser-based interactive exhibits and kiosk applications
- Data visualisations embedded in the museum website
- Generative art pieces that visitors can remix or influence via URL parameters
- Educational coding workshops using the browser — no software installation needed

## p5.js + ML5.js

ml5.js is a friendly machine learning library built on top of p5.js that wraps TensorFlow.js models (including MediaPipe). Together they enable body tracking, image classification, and text generation entirely in the browser.
