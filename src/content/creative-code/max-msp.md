Max (formerly Max/MSP, now often just "Max") is a visual programming environment by Cycling '74 for music, audio, and interactive media. It shares the node-based paradigm of TouchDesigner and vvvv, but its roots are in music and audio processing — making it the strongest choice for any installation where sound and interaction are primary.

## Max vs TouchDesigner

Both are node-based and real-time. The key difference is where each tool started:
- **Max** began as a music tool → strongest for audio, MIDI, music synthesis, and sound-reactive systems
- **TouchDesigner** began as a visual tool → strongest for GPU rendering, video, and projection mapping

Many studios use both: TouchDesigner for visuals, Max for audio, connected via OSC.

## Max for Live

Max integrates directly with Ableton Live as "Max for Live" — custom audio/MIDI devices and effects built in Max run inside Ableton. This makes it a powerful tool for live music performances and sound installations that use Ableton as a backbone.

## Use Cases

- Real-time audio synthesis and processing for sound installations
- Interactive sound pieces that respond to visitor movement, sensors, or data
- Show control systems that use audio timing and analysis to trigger events
- Custom MIDI controllers and hardware instruments built on Max
