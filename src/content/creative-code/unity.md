Unity is a cross-platform real-time 3D engine widely used in interactive art, museum installations, VR/AR experiences, and spatial computing. It sits between TouchDesigner (effects/patching) and Unreal (high-end rendering) — it is more accessible than Unreal but more structured than TouchDesigner, and has a large community producing open-source museum and XR work.

## Why Unity for Installations

Unity's greatest strength for museum work is its cross-platform reach and the breadth of its ecosystem. The same project can target Windows (PC display), Android (tablets, standalone VR headsets like Meta Quest), iOS, WebGL, and more. It is the dominant platform for XR experiences.

## Key Features

- **C# scripting:** Clean, typed language with a large learning community
- **Unity Visual Scripting (Bolt):** Node-based scripting without code, similar to Blueprints
- **Universal Render Pipeline (URP):** Optimised rendering for interactive/XR work
- **High Definition Render Pipeline (HDRP):** Photorealistic rendering for high-spec hardware
- **XR Toolkit:** Standardised VR/AR interaction framework across headsets
- **Timeline:** Sequencing tool for cinematic and scheduled content

## Museum Use Cases

- Interactive gallery kiosks and touch-screen experiences
- VR and AR experiences using Meta Quest or HoloLens
- Mixed-reality performances combining live sensors with 3D content
- WebGL browser-based companion experiences
- Educational simulations and reconstructions
